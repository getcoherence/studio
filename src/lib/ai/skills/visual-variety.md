# Skill: Visual Variety

## When to Apply
When generating any scene plan. The #1 failure mode is "template sameness" — two
different companies' videos looking identical. Every decision should push toward
visual uniqueness.

## Rules

### Scene Type Diversity
1. **Never use the same scene type more than twice** in a single video.
2. **Mix categories.** Every plan should include scenes from at least 3 of:
   text, data, visual, social, cinematic, cta.
3. **Use the variant field.** When a scene type supports variants (e.g.,
   metrics-dashboard has counter-row, bar-chart, pie-radial, ticker-tape),
   always specify one. Never rely on the default.

### Animation Diversity
4. **No 3+ consecutive scenes with the same animation.** If you used "chars"
   twice, the next scene must use something else.
5. **Match animation to content energy.** Calm content → words, scale, blur-in.
   Energetic content → bounce, staccato, glitch, split. Data → chars, typewriter.
6. **Use at least 3 different animation types** per video.

### Background Diversity
7. **Alternate dark and light backgrounds.** If you use 2 dark scenes in a row,
   the next must be light (or vice versa).
8. **Use backgroundEffect on at least 40% of scenes.** Particle effects, animated
   gradients, and overlays add motion without requiring content changes.
9. **Vary background effects.** Don't use the same backgroundEffect twice in a row.

### Transition Diversity
10. **Use at least 2 different transition types.** Mix energy: fade for calm moments,
    striped-slam or zoom-punch for impact moments.
11. **Match transition energy to the emotional beat.** Problem → solution transition
    should be dramatic (color-burst, diagonal-reveal). Between similar scenes, use
    subtle (fade, slide).

### Brand Differentiation
12. **Use the brand accent color aggressively.** Accent words, transition colors,
    backgroundEffect colors, and chart colors should all derive from the brand.
13. **Vary font families.** Use sans-serif for body, but consider condensed for
    impact scenes and serif for cinematic/editorial scenes.

## Common Mistakes
- Generating 8 scenes that all use "hero-text" with "chars" animation
- Every background being "#0a0a1a" (dark solid)
- No background effects on any scene
- Using only "fade" transitions
- Ignoring the accent color (everything defaults to blue)
